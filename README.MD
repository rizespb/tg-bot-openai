grammy - пакет для создания ТГ ботов

openai - пакет, который позволяет разработчикам подключать искусственный интеллект от OpenAI в свои приложения - единая точка входа для работы с LLM, поддерживающими openai

По-умолчанию, бот использует API openRouter, т.к. он бесплатный

Деплой осуществляется через аналог git - sourcecraft То есть на sourcecraft.dev хранится клон репозитория https://github.com/rizespb/tg-bot-openai/

Доступ к sourcecraft.dev осуществляется на основе Персонального токена доступа, который действует до 1 апреля 2027

Конфигурация ci .sourcecraft/ci.yaml целиком взята из шаблона sourcecraft template-tg-bot
